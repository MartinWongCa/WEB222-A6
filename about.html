Welcome to my website

I will be applying some CSS layout techniques in this website, this website contents 5 pages, home, Product Registration, about, honesty and Reflection. All 5 pages will share the same css file to maintain consistent layout across the pages.

In home page, I have apply difference CSS and DOM techniques,
  • The header and navigation bar are fixed at the top position, those will be keep when scrolling down the page.
  • The CSS Grid is generated by DOM, it will default load 4 images. Also, it will add 4 cards for every click on "Load More" button.
  • The images are use CSS Grid for display. Images will display in 3 or 4 cards per row depends on the window size.
  • When the mouse hover on top of the image, the background color of the pointed card will be change.
  • Added some tags and buttons to the card, pressing on the button will transfer to original image webpage.
